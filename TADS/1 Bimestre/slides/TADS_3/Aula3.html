Embutindo o código na página

  • • A maioria dos programas PHP produz HTML como sua saída:

         páginas dinâmicas; respostas aos envios de formulários HTML; etc.

  • • Um programa de PHP embutido é um arquivo que contém uma mistura de HTML e PHP. 

                 
<?php
print "<!DOCTYPE html PUBLIC \"-//W3C//DTD XHTML 1.1//EN\"\n";
print " \"http://www.w3.org/TR/xhtml11/DTD/xhtml11.dtd\">\n";
print "<html xmlns=\"http://www.w3.org/1999/xhtml\">\n";
print "  <head>\n";
print "    <title>Hello!</title>\n";
...
?>
  • Imprimir código HTML com print ou echo torna o código bagunçado e propenso a erros:
    • •O html deve vir entre aspas e os caracteres especiais devem ser tratados, e.x. \"
    • •Quebras de linha no código \n devem ser inseridas a cada nova linha

 $name = array();                         # Criando um array

$name[index] = value;		# setando o valor de uma posição
$name[] = value;			# anexa
$a = array();     # Array vazio (length 0)
$a[0] = 23;       # Armazena o inteiro 23 na posição 0 (length 1)
$a2 = array("algumas", "strings", "em", "um", "array");

  • •para anexar, use chaves sem especificar uma posição

  • •o tipo de array não é espeficicado; tipos podem ser misturados

Um array pode ser criado com o construtor de linguagem array(). Ele pega um certo número de pares separados por vírgula chave => valor . 

array(  chave =>  valor
     , ...
     )
// chave pode ser tanto string ou um integer
// valor pode ser qualquer coisa 
<?php
$arr = array("foo" => "bar", 12 => true);

echo $arr["foo"]; // bar
echo $arr[12];    // 1
?>
